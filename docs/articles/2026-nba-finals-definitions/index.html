Metrics & Data Pipeline

Precise definitions, formulas, and on-court meaning for the 14 radar axes used across the per-team decks and the comparison page, plus the full data pipeline. Order matches the radar (clockwise from TS% at the top).

Every originally "lower is better" raw metric is published as its positive opposite (Ball Security = 100−TOV%, Def. Stops = 120−DefRtg, Foul Discipline = 6−PF/G, Matchup Suppression = 100−opp FG%), so "further from center = better" holds. The 14 rows below are exactly what the radar plots, in clockwise order; the last four defensive axes (BLK% / STL% / Forced TOV% / Matchup Suppression) come from real per-game Finals data via nba_api.

Data Acquisition Pipeline

1 · Source & scraping

Official `stats.nba.com` only, via `nba_api`: `LeagueGameLog` (playoff games), `BoxScoreTraditionalV3`, `BoxScoreMatchupsV3`, `PlayByPlayV3` — covering the 5 game 2026 Finals (Knicks 4-1 Spurs) plus each team's full playoff run. Polite: 0.6–1.2s jittered backoff, raw JSON cached under `data/raw/`, analysis reads cache only.

2 · Three baselines

Each player gets three series, all plotted on the radar: (i) this Finals, game-by-game aggregated; (ii) the player's full-playoff average (Finals included, gray line); (iii) team-wide Finals average — a player / self / team triangle.

3 · Metric computation

10 offense/impact axes + 4 defense axes, all published positive (Ball Security = 100−TOV%, Def. Stops = 120−DefRtg, Foul Discipline = 6−PF/G, Matchup Suppression = 100−opp FG%). Defense four: BLK% / STL% from the box score, Forced TOV% by attributing steals to the defender via `PlayByPlayV3`, Matchup Suppression from `BoxScoreMatchupsV3` per-matchup aggregation.

4 · Normalization & repro

Radar uses per-axis min/max across the full set so any two players are area-comparable; tables keep raw values. The collection and build scripts (`scrape_defense.py` / `bake14.py` / `build_*.py`) re-run independently to reproduce everything.

All scripts live under src/skills/nba-finals-radar-publishing/ for independent reproduction.
