# Codex Instruction Notes

This folder summarizes the instruction layers that are visible and operationally relevant in the current local Codex environment.

## Scope

These notes are intentionally limited to:

- local visible instruction files
- observable local Codex config and runtime preferences
- project-local permission settings

They do not attempt to restate hidden platform prompts that are not owned by this repository.

## Files

- `senior-engineer-mentor-mode.md`
- `documentation-sync-and-completion-gate.md`
- `local-websearch-permission.md`
- `codex-runtime-preferences.md`

## Why keep this here

- document how the local Codex environment is actually being steered
- preserve rationale, trade-offs, and usage examples
- make future cleanup or migration easier
- avoid treating instruction files as mysterious black boxes
